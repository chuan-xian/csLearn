1. 将函数重新定义写一遍

2. 
```python
# 1. __str__()函数：
# 该函数是在调用print()打印对象的时候自动调用的
# 是给用户用的，是一个描述对象的方法。
# 就是print(per)时会执行该函数
__str__():
    pass

# 2. __repr__()函数：
# 该函数是给机器用的，在python解释器(终端)
# 里面直接敲对象名再回车后调用的方法
__repr__():
    pass


# 3. 在有repr而没有str时，str = repr
# 所以平时我们常用str函数

# 4.直接打印per对象时会输出类型和地址
# 而不会输出它的属性和行为
# 但是当我们要打印对象时
# 可以把要打印的信息写在str函数中
# 然后直接print(per)就可以
# 相当于print(per.__str__())
```
