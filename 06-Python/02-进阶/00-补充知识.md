# 装饰器(难点)

1. 本质
> 其实就是定义内外函数，把原函数作为参数传递给外函数，返回内函数的地址
> 
>   在内函数中调用原函数，并且把参数传给原函数

2. 示例：
```python
def outer(func):
def inner(*args, **kwargs):
    # 添加要修改的功能
    func(*args, **kwargs)
    return inner

@outer	#@外函数
def 原函数(参数):
    print("这是原函数！")
```
