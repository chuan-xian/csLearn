# 读文件

1. 通过文件操作符去操作文件的读写
> 过程：
> 
> 1. 打开文件
> 
> 2. 读写文件
> 
> 3. 关闭文件

2. 打开文件
```python
1. open(path, flag [,encoding] [,errors])
path：要打开文件的路径

flag：打开方式
    r：以只读的方式打开文件，文件的描述符放在文件的开头
    rb：以二进制格式只读打开一个文件，文件描述符放在文件的开头
    r+：打开一个文件用于读写，文件描述符放在文件的开头
    w：打开一个文件仅写入，文件存在则覆盖，不存在则新创建文件
    wb：打开一个文件仅写入二进制，文件存在则覆盖，不存在则新创建
    w+：打开一个文件用于读写，文件存在则覆盖，不存在则新创建文件
    a：打开一个文件追加内容，如文件存在，文件描述符将会放到末尾
    a+：打开一个文件用于追加内容，文件描述符放到末尾
# 文件描述符在哪，就是从哪里开始读取数据，常用的有r,rb,w,wb

encoding：编码方式，utf-8等

errors：ignore

f = open(path, "r")
```

3. 读取文件
```python
1. 读取文件全部内容：
# 不常用，如果文件过大就不可用
str = f.read()

2. 读取指定字符数：
# 读取十个字符，中文也算一个字符
str = f.read(10)

3. 读取整行：
# 读取一行，包括"\n"
str = f.readline()

4. 读取指定字符数：
# 读取10个字符，和read(10)一样
str = f.readline(10)

5. 读取所有行，并返回一个列表：
# 读取所有的行，每一行作为列表的一项元素返回
list = f.readlines()

6. 读取给定数字的字符，按行输出，最后一行也完整输出：
# 假设第一行11个字符，第二行15个字符，那么完整输出两行
list = f.readlines(20)

7. 修改描述符的位置：
# 将描述符置于第n个字符处
f.seek(n)
```

4. 关闭文件
```python
# 注意：在打开文件，然后read()一个文件后
# 文件描述符会移动到相应的位置
# 必须关闭才能重新把文件描述符置于开头
f.close()
```

5. 完整过程
```python
# 先打开文件到f1
# 如果读取失败，那么执行finally语句
# 如果打开失败，finally语句中的判断会是假，不需要执行f1.close()
try:
    f1 = open(path, "r", encoding="utf-8")
    print(f1.read())
finally:
    if f1:
        f1.close()

# 更简洁的写法：
# 正常读取会自动关闭文件
# 读取失败也会关闭文件
# 打开失败就不需要关闭文件
with open(path, "r", encoding="utf-8") as f2:
    print(f2.read())

#上面两种写法完全等价
```
